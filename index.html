What is 
StockHunt?

StockHunt is web and mobile application making all the hard work for you and presenting analysis for day trading and value investing with a user friendly interface.

Dont spend ​enormous time for stock analysis. Just live your life, StockHunt makes all technical and fundamental analysis and informs you when necessary.

OFFERINGS

WHAT WE DO OFFER

Easy To Use

StockHunt aims to provide convenient and easy to use interface  for investors to help them in ​analyzing companies easily.

Real Time Data

StockHunt contains historical and real time stock price of companies and also their financial information like balance sheets and income statements.

Wide Range Of Strategies

StockHunt contains 9 different technical indicators, 61 pattern types in day trading module and 13 different metrics in value investing module.

Cross-Platform Support

StockHunt supports web application, android and iOS devices. Notifications are send to users' all available devices.

Scalable

With high scalable structure, wide range of users can concurrently use the software.

Secure

StockHunt uses Django Framework which provides protection against common attacks like SQL injection, Cross site scripting (XSS), CSRF and Clickjacking.

Value Investing

Value investing is a type of  investment strategy that involves determining stocks that appear to be trading for less than their intrinsic or book value. StockHunt contains most of the value investing strategies suggested  by Benjamin Graham in his famous book "The Intelligent Investor".

  • Analyze companies with respect to 13 value investing metrics
  • Compare companies with respect to value investing scores.
  • Provide description of metrics for novice investors.

Pattern Recognition

Transitions between rising and declining trends are frequently suggested by price patterns in technical analysis. A pricing pattern is defined as a recognized pattern of price movement that can be determined using a sequence of trendlines and/or curves.

  • Identify 61 different type of patterns
  • Calculate identified pattern's correlation score.

Future Price Prediction

  • Predicts closing price of next day using machine learning model trained with historical stock price data .

Notifications

  • Sends notification to users in case significant trend change occurs in prechosen technical indicators for companies in users' watchlist.
  • Sends notification in case prechosen pattern type is identified in the stock price chart.

Hot Stock Suggestions

  • Recommend stocks receiving strong buy signals from majority of indicators and patterns. 
  • Daily list of companies ranked by their buy signals
